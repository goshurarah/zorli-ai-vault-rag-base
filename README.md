# Zorli AI Vault

Zorli AI Vault is a secure, AI-powered digital vault for individuals, teams, and enterprises. It allows users to safely store credentials, manage sensitive documents, and interact with their data using AI-powered Retrieval-Augmented Generation (RAG). Built for web and mobile, Zorli combines security, intelligence, and usability in a single platform.
